# ECOMMERCE-SYNTAXWEAR

## Descrição

Este é o site de e-commerce **SyntaxWear**, uma loja online especializada em tênis e sneakers. O projeto apresenta uma interface moderna e responsiva para compra de calçados, com seções para categorias de produtos, destaques e navegação intuitiva.

## Funcionalidades

- **Página Inicial**: Apresenta um banner hero com chamada para ação, categorias de produtos (Casual, Esporte, Moderno) e grid de produtos.
- **Navegação**: Menu responsivo com categorias principais e links de acesso rápido (Minha Conta, Carrinho, Ajuda).
- **Design Responsivo**: Otimizado para desktop e dispositivos móveis.
- **Estrutura Modular**: CSS organizado em componentes para fácil manutenção.

## Tecnologias Utilizadas

- **HTML5**: Estrutura semântica da página.
- **CSS3**: Estilização com variáveis, reset e componentes modulares.
- **Imagens SVG**: Para logos e ícones, garantindo escalabilidade.

## Estrutura do Projeto

```

├── index.html                 # Página principal do site

│   ├── reset.css              # Reset de estilos padrão

│   └── components/
│       ├── footer.css         # Estilos do rodapé
│       ├── header.css         # Estilos do cabeçalho
│       ├── hero.css           # Estilos da seção hero
│       ├── product-category.css # Estilos das categorias de produto
│       └── product-grid.css   # Estilos do grid de produtos
└── images/
    ├── banners/               # Imagens de banners
    ├── icons/                 # Ícones SVG
    ├── logo/                  # Logo da marca
    └── products/              # Imagens dos produtos
```

## Como Executar

Como este é um site estático, não há necessidade de instalação de dependências ou servidor. Basta abrir o arquivo `index.html` em qualquer navegador web moderno.

1. Clone ou baixe o repositório.
2. Navegue até a pasta do projeto.
3. Abra o arquivo `index.html` no seu navegador.
